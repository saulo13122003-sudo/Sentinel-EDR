🛡️ THE SENTINEL - Advanced Endpoint Protection (EDR)

Sentinel es una herramienta de vigilancia y seguridad en tiempo real desarrollada en Python. Su objetivo es proteger sistemas mediante la monitorización activa de directorios, aplicando un motor de puntuación de riesgos (Scoring) para identificar y neutralizar amenazas que los antivirus convencionales podrían pasar por alto.

Este proyecto documenta mi evolución como estudiante de ciberseguridad, desde un script de automatización básico hasta un sistema de defensa con persistencia en base de datos.
